# momodo

마스토돈(Mastodon) 4.6을 기반으로 소규모 폐쇄형 커뮤니티 — 특히 자캐(OC) 교류·롤플레이 서버 — 에 맞게 손본 커스텀 포크입니다. 특정 서버를 굴리려고 만든 것을 코드만 공개해 둔 것이라, 범용 배포판이라기보다 "이렇게 고쳐 쓴 마스토돈"의 공개본에 가깝습니다.

- 베이스: [mastodon/mastodon](https://github.com/mastodon/mastodon) v4.6.x (패치 릴리스 수시 병합)
- 원본 마스토돈의 일반적인 설명과 설치 문서는 공식 저장소를 참고하세요.

한 줄로 요약하면 **밖에서는 안 보이고, 안에서는 트위터처럼 쓰는 마스토돈**입니다.

---

## 기존 마스토돈과 다른 점

| | 마스토돈 순정 | 이 포크 |
|---|---|---|
| 비로그인 접근 | 프로필·게시물·타임라인 열람 가능 | 전부 차단, 로그인 화면으로 이동 |
| 연합(다른 서버 연결) | 기본 열림 | 기본 닫힘 (운영자가 허용한 서버만) |
| 게시물 공개범위 | 공개 / 미등재 / 팔로워 / 멘션 | "공개" 없음 — 전체 공개(서버 안에서만) / 팔로워 / 개인 멘션 |
| 기본 테마 | 마스토돈 기본 | 트위터풍(파랑새) — 마스토돈 테마도 선택 가능 |
| 계정 | 브라우저당 1계정 | 여러 계정 등록 후 전환(부계 운용) |
| 글자 꾸미기 | 없음 | 디스코드식 문법(굵게·기울임·스포일러 등) |
| 게시 예약 | API만 있고 화면 없음 | 작성창에서 바로 예약 + 예약 목록 관리 |
| 그룹 대화 | 없음 | 멤버 전용 실시간 방(그룹 메시지) |

## 좌측 메뉴

추가되거나 성격이 바뀐 메뉴만 적습니다. 나머지(홈·알림·북마크·좋아요·리스트·환경설정)는 순정과 같습니다.

| 메뉴명 | 설명 |
|---|---|
| 개인 멘션 | DM 대화 목록. 순정의 "개인적인 멘션"과 같은 기능입니다. |
| 답장 안 한 메시지 | 최근에 받은 멘션 100개 중 아직 답장도, 좋아요도, 부스트도 하지 않은 것만 모아 보여줍니다. 여기서 바로 답장할 수 있고, 답장·좋아요·부스트 중 하나라도 하면 목록에서 즉시 사라집니다. |
| 그룹 메시지 | 멤버 전용 실시간 방. 아래 "그룹 메시지(방)" 항목 참고. |
| 퍼블릭 모아보기 | 서버 전체의 "전체 공개" 게시물이 실시간으로 흐르는 타임라인. 순정의 라이브 피드에 해당합니다. |
| 예약 메시지 | 예약해 둔 게시물 목록(시간순). 취소하거나, 내용을 작성창으로 다시 불러와 고쳐서 재예약할 수 있습니다. |
| 계정 전환 | 이 브라우저에 등록된 계정 목록. 계정 추가·전환·제거·전체 로그아웃. |

## 글쓰기

### 공개범위

작성창 왼쪽 위 드롭다운에서 선택합니다.

- **전체 공개** — 서버 구성원 전체에게 공개. 퍼블릭 모아보기에 실시간으로 흐릅니다. 이름은 전체 공개지만 서버 바깥(다른 마스토돈 서버, 검색엔진)에는 나가지 않습니다.
- **팔로워** — 내 팔로워에게만.
- **개인 멘션** — 글에 멘션된 사람에게만. 사실상 DM입니다.

순정에 있는 "공개"(연합 전체 공개)는 이 포크에서 선택할 수 없습니다.

### 글자 효과

디스코드와 같은 문법을 그대로 씁니다. 게시하면 적용된 모양으로 보이고, 수정할 때는 입력한 문법 그대로 돌아옵니다.

| 입력 | 결과 |
|---|---|
| `**굵게**` | **굵게** |
| `*기울임*` 또는 `_기울임_` | *기울임* |
| `__밑줄__` | 밑줄 |
| `~~취소선~~` | ~~취소선~~ |
| `\|\|스포일러\|\|` | 가려진 상태로 게시되고, 읽는 사람이 클릭하면 보입니다 |
| `` `코드` `` | 고정폭 코드 |
| ```` ```여러 줄``` ```` | 코드 블록 |
| `\*` | 문법 기호를 글자 그대로 쓰고 싶을 때 |

문법이 잘 기억나지 않으면 작성창 아래 **글자 서식** 버튼(Ⓐ 모양)을 누르세요. 목록에서 고르면 커서 위치에 문법이 들어가고, 텍스트를 드래그해 둔 상태라면 그 부분을 감쌉니다.

### 게시 예약

작성창 아래 **시계 버튼**을 누르면 시간을 고를 수 있습니다(30분 뒤·1시간 뒤 같은 버튼, 또는 직접 입력. 최소 5분 이후). 시간을 정하면 게시 버튼이 "예약"으로 바뀝니다.

- 예약한 글은 좌측 **예약 메시지** 메뉴에서 확인합니다.
- 내용 수정은 "취소 후 다시 쓰기" 방식입니다 — 수정 버튼을 누르면 예약이 취소되고 내용이 작성창으로 돌아옵니다. 고쳐서 다시 게시하거나 재예약하세요. 첨부파일은 다시 올려야 합니다.
- 답글도 예약할 수 있습니다. 발행되면 원래 글의 타래에 정상적으로 붙습니다. 발행 전에 원글이 삭제되면 그 예약은 발행되지 않고 조용히 사라집니다.
- 그룹 메시지(방) 글은 예약할 수 없습니다.

## 계정 전환 (부계 운용)

좌측 **계정 전환** → **기존 계정 추가**로 다른 계정에 로그인해 두면, 이후에는 목록에서 클릭 한 번으로 전환됩니다.

- 지금 쓰는 계정을 로그아웃하지 않고 추가합니다.
- 등록 정보는 이 브라우저에만 저장됩니다. 다른 기기·다른 브라우저에는 따라가지 않습니다.
- 캐시를 지우거나 브라우저를 재시작해도 유지됩니다. 쿠키를 지우면 전부 로그아웃됩니다.

## 그룹 메시지 (방)

디스코드 채널 비슷한 멤버 전용 공간입니다.

- 모든 방은 초대제입니다. 방을 만든 사람(방장)이 멤버를 초대합니다.
- 방에 쓴 글은 그 방 안에서만 보입니다. 프로필·타임라인·검색 어디에도 나오지 않고, 게시물 수에도 잡히지 않습니다.
- 멘션이 생기지 않으므로 알림도 가지 않습니다. 실시간으로 새 글이 올라옵니다.
- 글을 클릭해도 상세보기로 이동하지 않습니다(답글·부스트 없음). 첨부는 이미지만 됩니다.
- 방에서 나가려면 방 화면의 **방 나가기**를 누르세요. 나간 뒤에도 그동안 쓴 글은 방에 남지만, 방을 다시 볼 수는 없습니다. 방장은 나갈 수 없습니다(방이 관리자 없이 남는 것을 막기 위해서입니다).

## 테마와 다크 모드

**환경설정 → 외관**에서 테마(트위터풍 / 마스토돈)와 색상 모드(밝게 / 어둡게 / 자동)를 각각 고릅니다. 기본값은 트위터풍입니다.

## 알아둘 것

- 알림 이메일은 기본적으로 전부 꺼져 있습니다. 받고 싶으면 환경설정에서 항목별로 켜세요.
- "발견하기" 계열 기능(유행, 프로필 추천 등)은 꺼져 있거나 제거돼 있습니다.
- 스포일러 문법(`||…||`)은 이 포크끼리만 완전하게 동작합니다. 연합을 열어 다른 순정 서버로 글이 나가면, 그쪽에서는 가려지지 않은 글자로 보입니다.
- 알림 메일·푸시 알림 미리보기에는 글자 효과 문법이 입력한 기호 그대로 보입니다.
- 좋아요를 취소하면 그 글은 "답장 안 한 메시지"에 다시 나타납니다.

## 서버 운영자용

### 설치

공식 [설치 문서](https://docs.joinmastodon.org/admin/install/)를 그대로 따르면 됩니다. 저장소만 이곳으로 바꾸세요.

```bash
git clone https://github.com/momodoho/momodo-custom-mastodon.git live
cd live && git checkout main
```

폐쇄형 동작(비로그인 차단, 연합 닫힘)은 전부 코드 기본값이라 따로 설정할 환경변수가 없습니다. 설치 마법사에서 도메인·DB·Redis·SMTP만 정하면 됩니다. web·sidekiq·streaming 세 서비스를 모두 띄우세요 — 예약 발행은 sidekiq이, 실시간 갱신과 그룹 메시지는 streaming이 담당합니다.

연합을 열고 싶으면 `LIMITED_FEDERATION_MODE=false`를 주거나, 관리 화면에서 허용 도메인을 추가하면 됩니다.

### 관리 화면에 추가된 것

- **관리 → 서버 설정 → 외관** — 로고, 로그인 화면 배경, 앱(타임라인) 배경을 웹에서 업로드. 테마별·밝음/어둠별로 따로 지정합니다.
- **관리 → 중재 → 그룹 메시지** — 방 목록·멤버 확인, 멤버 퇴장, CSV 내보내기.
- **관리 → 계정 → (계정) → 게시물** — DM 포함 전체 게시물 열람, 메시지 유형 필터, 대화 상대 검색. 폐쇄형 운영을 전제로 한 중재 도구입니다.
