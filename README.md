# momodo

자캐(오리지널 캐릭터) 교류 커뮤니티를 돌리려고 마스토돈(Mastodon)을 손본 버전입니다. 트위터를 쓰던 사람들이 낯설지 않게, 그리고 커뮤니티 안에서 쓴 글이 바깥으로 새지 않게. 이 두 가지가 목표였고, 나머지 수정사항은 대부분 여기서 따라 나온 것들이에요.

특정 서버를 운영하려고 만든 걸 코드만 공개해 둔 거라, 아무 서버에나 바로 얹는 범용 배포판은 아닙니다. 다만 같은 목적(소규모 폐쇄 커뮤니티)이라면 그대로 가져다 써도 됩니다.

원본 마스토돈이 뭔지 궁금하면 [공식 저장소](https://github.com/mastodon/mastodon)를 보세요. 이 문서는 "원본과 뭐가 다른지"만 다룹니다.

---

## 일단 제일 큰 차이

**로그인 안 하면 아무것도 안 보입니다.** 프로필도, 글도, 검색도 전부요. 링크를 밖에 공유해도 로그인 화면만 뜹니다. 다른 마스토돈 서버와의 연결(연합)도 기본으로 꺼져 있어서, 여기 쓴 글은 이 서버 안에서만 돕니다. 검색엔진에도 안 잡혀요.

그래서 글 공개범위에 "공개"가 없습니다. 제일 넓은 게 **전체 공개**인데, 이름과 달리 이 서버 회원 전체까지만입니다. 그 아래로 **팔로워**, 그리고 멘션된 사람만 보는 **개인 멘션**(사실상 DM)이 있어요.

겉모습은 트위터풍 테마가 기본입니다. 마스토돈 원래 모습이 좋으면 환경설정 → 외관에서 바꿀 수 있고, 밝게/어둡게도 거기서 고릅니다.

## 글 꾸미기

디스코드 쓰던 사람은 따로 배울 게 없습니다. 같은 문법이 그대로 먹혀요.

| 이렇게 쓰면 | 이렇게 보입니다 |
|---|---|
| `**굵게**` | **굵게** |
| `*기울임*` | *기울임* |
| `__밑줄__` | 밑줄 |
| `~~취소선~~` | ~~취소선~~ |
| `\|\|스포일러\|\|` | 가려져서 올라가고, 누르면 보임 |
| `` `코드` `` | 고정폭 글씨 |

기호가 헷갈리면 작성창 아래 **글자 서식** 버튼을 누르세요. 목록에서 고르면 커서 자리에 기호가 들어가고, 글자를 드래그해 놓고 누르면 그 부분을 감싸줍니다.

스포일러는 자캐 커뮤니티에서 쓸 일이 많아서 넣었습니다. 반전 내용이나 잔인한 묘사 같은 걸 가려서 올릴 때 쓰세요. 읽는 쪽에서 클릭해야 열립니다.

## 예약 글

작성창 아래 시계 모양 버튼을 누르면 시간을 고를 수 있어요. 30분 뒤, 1시간 뒤 같은 버튼도 있고 직접 날짜와 시간을 정할 수도 있습니다(지금부터 5분 이후부터 가능). 시간을 정하면 게시 버튼이 "예약"으로 바뀝니다.

예약해 둔 글은 왼쪽 메뉴의 **예약 메시지**에서 봅니다. 여기서 취소할 수 있고, 고치고 싶으면 "수정" 을 누르세요. 예약이 취소되면서 내용이 작성창으로 돌아오니까, 고쳐서 다시 예약하면 됩니다. 첨부한 그림은 같이 안 돌아오니 다시 올려야 해요.

답글도 예약이 됩니다. 시간이 되면 원래 타래에 제대로 붙어요. 다만 예약해 둔 사이에 원글이 지워지면 그 예약은 그냥 발행이 안 되고 사라집니다. 엉뚱한 데 홀로 올라가는 것보다는 나아서 그렇게 해뒀습니다.

## 부계정 굴리기

자캐 커뮤니티는 한 사람이 계정 여러 개 굴리는 게 보통이라, 트위터처럼 계정을 등록해두고 바꿔가며 쓸 수 있게 했습니다.

왼쪽 메뉴 **계정 전환** → **기존 계정 추가**로 다른 계정에 로그인해 두면, 그다음부터는 목록에서 누르기만 하면 바로 전환됩니다. 지금 계정이 로그아웃되지 않아요.

등록 정보는 그 브라우저에만 남습니다. 폰이나 다른 컴퓨터에는 따라가지 않고, 캐시를 지워도 유지되지만 쿠키를 지우면 전부 로그아웃됩니다.

## 그룹 메시지 (방)

디스코드 채널 같은 멤버 전용 공간입니다. 합작이나 소규모 기획, 비공개 수다방 용도로 쓰세요.

- 방은 전부 초대제입니다. 방을 만든 사람이 멤버를 초대해요.
- 방에 쓴 글은 방 안에서만 보입니다. 프로필에도, 타임라인에도, 검색에도 안 나오고 게시물 수에도 안 잡혀요.
- 멘션이 안 생기니 알림도 안 갑니다. 방을 열어두면 새 글이 실시간으로 올라옵니다.
- 글을 눌러도 상세보기로 안 넘어갑니다. 답글이나 부스트 개념이 없는, 그냥 채팅에 가까운 공간이에요. 그림 첨부는 됩니다.
- 나가고 싶으면 방 화면의 **방 나가기**. 나가도 그동안 쓴 글은 방에 남지만, 그 방을 다시 볼 수는 없게 됩니다. 방장은 못 나갑니다(주인 없는 방이 생기는 걸 막으려고요).

## 답장 안 한 메시지

받은 멘션이 쌓이다 보면 뭘 답했고 뭘 놓쳤는지 헷갈리죠. 왼쪽 메뉴의 **답장 안 한 메시지**를 열면, 최근에 받은 멘션 100개 중에서 아직 아무 반응도 안 한 것만 모아서 보여줍니다.

여기서 바로 답장할 수 있고, 답장하거나 좋아요를 찍거나 부스트를 하면 그 자리에서 목록에서 빠집니다. 반대로 좋아요를 취소하면 다시 나타나요. "처리했다"의 기준이 그 세 가지라서 그렇습니다.

## 자잘하게 알아둘 것

- 알림 이메일은 기본으로 전부 꺼져 있습니다. 필요하면 환경설정에서 항목별로 켜세요.
- 유행(트렌드)이나 프로필 추천 같은 "발견하기" 기능은 꺼놨습니다. 폐쇄 커뮤니티에는 필요가 없어서요.
- 나중에 연합을 열어서 다른 마스토돈 서버와 연결하게 되면, 스포일러 문법은 저쪽 서버에서는 안 가려진 채로 보입니다. 저쪽은 순정 마스토돈이라 이 기능을 모르거든요.
- 푸시 알림이나 알림 메일 미리보기에는 글자 효과가 적용되지 않고 `**이런**` 기호가 그대로 보입니다.

## 서버를 직접 띄우고 싶다면

여기부터는 운영자 얘기입니다. 마스토돈 공식 [설치 문서](https://docs.joinmastodon.org/admin/install/)를 그대로 따라가되, 저장소만 이걸로 바꾸면 됩니다.

```bash
git clone https://github.com/momodoho/momodo-custom-mastodon.git live
cd live && git checkout main
```

폐쇄형 동작(비로그인 차단, 연합 차단)은 전부 코드 기본값이라 따로 켜줄 환경변수가 없습니다. 설치 마법사에서 도메인, DB, Redis, 메일 서버만 정하면 그대로 폐쇄 커뮤니티로 뜹니다. web, sidekiq, streaming 세 서비스는 다 띄우세요. 예약 글 발행은 sidekiq이 하고, 실시간 갱신이랑 그룹 메시지는 streaming이 합니다.

연합을 열고 싶으면 `LIMITED_FEDERATION_MODE=false`를 주거나 관리 화면에서 허용할 서버를 추가하면 됩니다.

관리 화면에 몇 가지가 더 있습니다. 서버 설정 → 외관에서 로고와 배경 이미지를 테마별로 업로드할 수 있고, 중재 메뉴에 그룹 메시지 관리(멤버 확인·퇴장)가 있고, 계정별 게시물 화면에서 DM까지 포함해 열람하고 대화 상대로 검색할 수 있습니다. 마지막 것은 폐쇄 커뮤니티 중재를 전제로 한 물건이니 운영 방침에 맞게 쓰세요.
